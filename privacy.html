Kebijakan Privasi

Terakhir diperbarui: 1 Januari 2024

📅 Berlaku sejak: 1 Januari 2024

1. Pendahuluan

Selamat datang di FlixID. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan layanan kami di chokierwin.github.io.

Dengan menggunakan FlixID, Anda menyetujui kebijakan privasi ini. Jika Anda tidak setuju, harap hentikan penggunaan layanan kami.

2. Informasi yang Kami Kumpulkan

FlixID tidak mengumpulkan data pribadi secara langsung. Namun, beberapa informasi teknis mungkin dikumpulkan secara otomatis:

  • Alamat IP dan informasi browser (melalui GitHub Pages)
  • Halaman yang dikunjungi dan waktu kunjungan
  • Data penggunaan agregat (bukan data pribadi)

⚠️ FlixID tidak meminta, menyimpan, atau memproses data pribadi seperti nama, email, atau kata sandi pengguna.

3. Penggunaan Data

Data teknis yang dikumpulkan digunakan untuk:

  • Memastikan website berfungsi dengan baik
  • Menganalisis trafik untuk meningkatkan layanan
  • Mendeteksi dan mencegah aktivitas berbahaya

4. Layanan Pihak Ketiga

FlixID menggunakan layanan pihak ketiga berikut:

  • The Movie Database (TMDB) — untuk data film, poster, dan informasi streaming. Lihat kebijakan privasi TMDB di themoviedb.org
  • YouTube (Google) — untuk menampilkan trailer film. Lihat kebijakan privasi Google di policies.google.com
  • GitHub Pages — untuk hosting website. Lihat kebijakan privasi GitHub di github.com/privacy
  • Google AdSense — untuk menampilkan iklan relevan. Google mungkin menggunakan cookie untuk personalisasi iklan.

5. Cookie

FlixID dapat menggunakan cookie untuk meningkatkan pengalaman pengguna. Cookie adalah file kecil yang disimpan di perangkat Anda. Anda dapat mengatur browser untuk menolak cookie, namun beberapa fitur website mungkin tidak berfungsi optimal.

Google AdSense menggunakan cookie untuk menampilkan iklan yang relevan berdasarkan kunjungan Anda sebelumnya ke website kami dan website lain.

6. Hak Cipta

Semua data film, gambar, dan informasi yang ditampilkan di FlixID bersumber dari The Movie Database (TMDB) dan digunakan sesuai dengan ketentuan penggunaan TMDB API. FlixID tidak mengklaim kepemilikan atas konten tersebut.

🎬 FlixID tidak menyimpan, mendistribusikan, atau menyiarkan konten film apapun. Kami hanya mengarahkan pengguna ke platform streaming resmi yang legal.

7. Keamanan Data

Kami berkomitmen untuk melindungi keamanan layanan kami. Namun, tidak ada metode transmisi data melalui internet yang 100% aman. Kami menggunakan langkah-langkah keamanan yang wajar untuk melindungi informasi.

8. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini sewaktu-waktu. Perubahan akan diberitahukan melalui pembaruan tanggal di bagian atas halaman ini. Penggunaan berkelanjutan atas layanan kami setelah perubahan dianggap sebagai persetujuan Anda.

9. DMCA & Hak Cipta

FlixID adalah platform agregator informasi film yang tidak menyimpan, menghosting, atau menyiarkan file video, film, atau serial TV apapun di server kami. Seluruh data film (judul, poster, sinopsis, rating) bersumber dari API publik The Movie Database (TMDB), dan trailer ditampilkan melalui embed resmi YouTube.

Tautan "Di Mana Nonton" pada FlixID hanya mengarahkan pengguna ke platform streaming berlisensi resmi seperti Netflix, Disney+, Prime Video, Viu, dan Vidio — bukan ke salinan film ilegal.

⚖️ Jika Anda adalah pemegang hak cipta dan merasa ada konten di FlixID yang melanggar hak Anda, silakan hubungi kami melalui halaman Kontak dengan menyertakan: (1) identifikasi karya yang dilindungi hak cipta, (2) lokasi/URL konten yang dimaksud, dan (3) informasi kontak Anda. Kami akan menindaklanjuti laporan yang valid secepatnya.

10. Hubungi Kami

Jika Anda memiliki pertanyaan tentang Kebijakan Privasi atau DMCA ini, silakan hubungi kami melalui halaman Kontak.
